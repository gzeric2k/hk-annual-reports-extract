# 港股年报目录模式

本文件给出常见港股年报目录格式，便于调试 `parse_toc_from_page()`。

## 模式1：标题在前，页码在后

示例：

```
公司簡介 ........ 6
財務概要 ........ 8
董事長報告 ...... 10
```

## 模式2：页码在前，标题在后

示例：

```
6 公司簡介
8 財務概要
10 董事長報告
```

## 模式3：章节编号 + 标题 + 页码

示例：

```
第一章 公司簡介 ........ 6
第二章 財務概要 ........ 8
```

## 模式4：两栏目录

左栏为页码，右栏为标题（提取后由 `<<<COLUMN_BREAK>>>` 分隔），再做 zip 配对。

## 模式5：空格打散字符

示例：

```
C O N T E N T S
公 司 簡 介
```

通过字符间空格归一化后再解析。
